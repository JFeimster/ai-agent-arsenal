# AGENTS.md

## Repository Scope
This repository is a scaffold and control layer for AI agent assets across `ai-tools/`, `directory/`, `showcase/`, `docs/`, and `data/`.

## Write-Safety Rules
- Only change files explicitly in the approved batch.
- Do not delete files.
- Do not move or rename files unless the batch explicitly requires it.
- Preserve imported source assets under `ai-tools/agents/source/`.
- Keep placeholder content operational and concise.

## Branch and PR Rules
- Use one branch per batch with the approved branch name.
- Open one draft PR per batch.
- Stop after each batch and wait for approval before continuing.

## Protected Areas
Do not modify repository structure, registries, schemas, source indexes, or imported upstream content outside approved batch scope without explicit approval.
