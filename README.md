# AI Agent Arsenal

AI Agent Arsenal is a centralized directory and development hub for CustomGPTs, Gemini Gems, Google AI Studio apps, prompt systems, fintech calculators, automation blueprints, and operator-grade AI tools.

This repo will organize experimental and production-ready AI tools across:

- CustomGPTs
- Gemini Gems
- Google AI Studio apps
- Flash UI static site variants
- Prompt systems
- Fintech calculators
- Funding and cash flow tools
- Affiliate growth tools
- Business credit tools
- Ecommerce finance tools
- Founder and operator workflows
