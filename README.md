# AI Agent Arsenal

Simplified repository structure:

```txt
README.md
directory/
ai-tools/
data/
docs/
showcase/
```

## Folder Meanings

- `directory/`: reserved public-facing static directory site.
- `ai-tools/`: AI tool source packages, assets, templates, prompts, schemas, skills, agents, and related libraries.
- `data/`: shared indexes, registries, manifests, and source imports.
- `docs/`: documentation, standards, governance, and import notes.
- `showcase/`: browsable UI concepts and demo builds (not production source of truth).
