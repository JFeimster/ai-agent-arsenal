## App Name

SOP Goblin

## Agent Source

`agents/07-sop-goblin.md`

## Buyer

Agencies, founders, operators, automation consultants, fractional COOs.

## User Promise

Paste messy notes, screenshots text, chat summaries, or workflow descriptions. Get a clean SOP with steps, owner, QA checklist, and troubleshooting notes.

## MVP Inputs

- Workflow name
- Messy notes
- Tools involved
- Desired output format
- Owner
- Known failure points

## MVP Outputs

- Objective
- Prerequisites
- Workflow Steps
- QA Checklist
- Troubleshooting
- Owner
- Version Notes

- Notion
- GitHub docs
- Google Drive
- n8n workflow docs
- Loom transcript notes
