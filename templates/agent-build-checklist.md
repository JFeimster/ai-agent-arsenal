# Agent Build Checklist

> Source: JFeimster/ai-agent-library. Imported/adapted for ai-agent-arsenal.


## Strategy
- [ ] Define buyer
- [ ] Define pain solved
- [ ] Define workflow trigger
- [ ] Define monetization path
- [ ] Define what good output looks like

## Knowledge Base
- [ ] Add source docs
- [ ] Add examples
- [ ] Add field maps or schemas
- [ ] Add brand/tone rules
- [ ] Add compliance or guardrail notes

## Prompting
- [ ] Write agent instructions
- [ ] Add rules
- [ ] Add conversation starters
- [ ] Add reusable prompts
- [ ] Add output format

## Testing
- [ ] Test with clean input
- [ ] Test with messy input
- [ ] Test missing data
- [ ] Test edge cases
- [ ] Confirm it refuses to invent facts

## Packaging
- [ ] Add agent bio
- [ ] Add setup steps
- [ ] Add screenshots or examples
- [ ] Add monetization notes
- [ ] Add future platform conversion notes
