# Suggested Notion Agent Library Database Schema

> Source: JFeimster/ai-agent-library. Imported/adapted for ai-agent-arsenal.


| Property | Type | Notes |
|---|---|---|
| Agent Name | Title | Main agent name |
| Slug | Text | File/repo slug |
| Platform | Multi-select | Notion AI, GPT, Claude, Gemini, n8n, Vercel |
| Category | Select | Funding Ops, CRM Ops, Content Ops, etc. |
| Buyer | Multi-select | Target customer |
| Pain Solved | Text | Core pain point |
| Monetization | Text | Product path |
| Status | Select | Draft, Testing, Active, Deprecated |
| Knowledge Files | Relation/File | Supporting docs |
| Integrations | Multi-select | GitHub, Vercel, Notion, Gmail, HubSpot, etc. |
| Owner | Person | Maintainer |
| Last Reviewed | Date | Maintenance date |
| Score | Number | Evaluation score |
| Repo File | URL/Text | Link to markdown file |
