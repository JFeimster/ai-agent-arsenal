# Claude Skill Format

Status: draft/proposed

Use this format for Claude-specific skills that rely on explicit task framing, constraints, and handoff-ready outputs.

## Required sections

## Objective
State the task in one sentence.

## Constraints
List file scope, tool assumptions, and non-goals.

## Inputs
List the required context.

## Outputs
Describe the expected artifact or decision.

## Review checks
- Scope is respected.
- Assumptions are explicit.
- Output is ready for implementation or review.
