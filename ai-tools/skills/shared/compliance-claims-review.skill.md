# Compliance Claims Review Skill

Status: draft/proposed

## Purpose
Screen public-facing copy for risky claims, guarantees, or unsupported statements.

## Inputs
- draft copy
- offer context
- required disclaimers

## Outputs
- flagged claims
- safer rewrites
- disclosure reminders

## Logic
1. Find guarantees, absolutes, and implied approvals.
2. Replace risky statements with bounded language.
3. Note where legal or compliance review is still needed.

## Guardrails
- This is not legal advice.
- Prefer conservative public-language rewrites.
