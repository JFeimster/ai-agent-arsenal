# Gemini Gem Skill Template

Status: draft/proposed

## Purpose
Template for a Gemini Gem or Google AI Studio prompt package.

## Inputs
- gem goal
- persona or audience
- reference material

## Outputs
- gem instruction block
- prompt starter set
- validation checklist

## Logic
1. Keep the Gem tied to one narrow job.
2. Define expected tone and refusal boundaries.
3. Supply a short starter prompt set for smoke testing.

## Guardrails
- Do not assume access to private Google Workspace data.
- Mark optional integrations as draft.
