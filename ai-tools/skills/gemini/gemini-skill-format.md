# Gemini Skill Format

Status: draft/proposed

Use this format for Gemini-specific skills, gems, and AI Studio app workflows.

## Required sections

## Use case
State the Gemini workflow this skill supports.

## Inputs
List the minimum setup and prompt context.

## Outputs
Describe the expected artifact or response pattern.

## Safety notes
- Keep platform assumptions explicit.
- Avoid tool claims that depend on unrevealed Google services or data.
