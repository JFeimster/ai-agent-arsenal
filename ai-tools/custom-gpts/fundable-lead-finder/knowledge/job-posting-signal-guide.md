# Job Posting Signal Guide

**File:** `job_posting_signal_guide.md`
**Owner GPT:** Fundable Lead Finder
**Purpose:** Maps open job roles to funding triggers. Hiring is one of the strongest, most timely signals that a local service business is scaling and likely to need capital. This guide tells the GPT how to read job posts.

> **How to use:** When job-posting enrichment is present, translate roles + volume into triggers and a score impact. The *type*, *number*, and *recency* of openings all matter.

---

## 1. Role → Signal Map

| Open Role | What It Signals | Funding Implication |
|---|---|---|
| **Field techs / installers** (HVAC, plumbing, electrical) | Rising service/install demand | Payroll + inventory/tools ramp |
| **Crews / laborers / helpers** (roofing, concrete, landscaping) | More job capacity, bigger backlog | Crew payroll + materials |
| **Drivers / CDL drivers** (trucking, delivery, towing) | Fleet utilization / expansion | Fleet finance + driver payroll |
| **Dispatchers** | Higher job/call volume to coordinate | Operational scale → working capital |
| **Sales reps / estimators / canvassers** | Growth push, pipeline expansion | Marketing/expansion capital |
| **Project managers / foremen** | Larger/more complex projects | Project financing, draw bridges |
| **Office / admin / CSR** | Operational scaling, more throughput | Working capital |
| **Mechanics / fleet maintenance** | Growing fleet to maintain | Fleet finance |
| **Branch / location manager** | New or planned location | Expansion capital |
| **Multiple roles at once** | Broad scaling / hiring spree | Strong multi-product opportunity |

---

## 2. Volume & Intensity

| Pattern | Interpretation | Score Lean (per rubric) |
|---|---|---|
| Single generic "always hiring" post | Weak/ambient signal | +3 |
| 1 specific recent opening | Modest growth | +3 to +6 |
| 2-3 simultaneous openings | Active scaling | +8 |
| 4+ openings / multi-role spree | Clear hiring spree | +10 |
| Repeated reposting of same role | High turnover OR sustained demand | Read in context; usually demand |
| Posting for a NEW location/branch | Expansion confirmed | +10 expansion |

---

## 3. Recency Rules

- **Last 30 days:** Strong, current signal — use confidently.
- **30-90 days:** Still relevant; slightly discount.
- **90+ days / undated:** Treat as soft; label "may be hiring."
- **"Always hiring" evergreen page:** Ambient only — minimal bonus.

---

## 4. Role Combos That Signal Specific Products

- **Drivers + mechanics** → fleet growing → **fleet financing**.
- **Crews + estimators** → backlog + pipeline → **working capital / project financing**.
- **Techs + dispatchers + CSRs** → service volume scaling → **working capital / inventory line**.
- **Sales reps + marketing roles** → growth push → **marketing/expansion capital**.
- **Branch manager + multiple field roles** → new location → **expansion capital**.

---

## 5. Cautions

- High turnover can mimic growth — if reviews show churn/management issues, temper the read.
- A single admin hire isn't a strong capital signal on its own.
- Don't invent job posts. Only infer from provided enrichment; if none, note hiring is "worth checking" rather than assumed.

---

## 6. Output Application

- **Trigger Analysis:** "3 open crew postings (confirmed) signal active scaling and near-term payroll load."
- **Score:** Apply the hiring bonus per volume/recency.
- **Outreach:** Lead with it — "Saw you're hiring drivers — adding trucks for a new contract?"

QUICK READ: role type → trigger ; count → intensity ; recency → confidence
field roles = demand | drivers = fleet | sales = growth | managers = expansion
