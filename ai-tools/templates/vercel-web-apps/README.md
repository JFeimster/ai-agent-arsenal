# Vercel Web App Templates

Source: JFeimster/ai-agent-library  
Imported/adapted for ai-agent-arsenal on 2026-05-30.

This folder contains reusable starter assets for converting agent specs into Vercel-hosted web apps.

Included templates:

- agent-config-template.json
- app-router-api-route-template.ts
- lead-capture-schema.json
- nextjs-agent-app-readme.md
- vercel-env-example.env

Notes:

- vercel-env-example.env contains placeholders only and no real secrets.
- Keep these templates generic; implementation-specific variants should live under showcase/builds/vercel-agent-apps/.
