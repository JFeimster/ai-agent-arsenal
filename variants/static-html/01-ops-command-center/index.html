Every single tool in this system must eliminate a distinct calculation bottleneck or administrative friction point. If it serves no workflow purpose, it goes in the trash.

An interactive agent without a path of action is just a database query. Every tool outputs distinct steps, immediate metrics, or an exportable framework.
